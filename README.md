# Octoman Read Later 📖

> **你的碎片信息整理站。** 浏览网页时随手保存，稍后集中阅读，不再让好内容淹没在无数标签页中。

---

## 🧠 为什么需要它？

每天我们都在浏览大量网页——一篇深度文章、一个实用教程、一份工作文档……当时没时间细看，于是打开一个新标签页"稍后阅读"。

结果呢？

- 🔴 标签页越开越多，Chrome 越来越卡
- 🔴 想找之前看到的那篇文章，翻遍所有标签页也找不到
- 🔴 好不容易找到了，又不敢关——怕再也找不回来

**Octoman Read Later** 就是为了解决这个问题而生的。它不是一个"又一项待办"，而是一个让你安心关闭标签页、随时回看、集中管理阅读内容的工具。

## 🎯 核心用途

| 场景 | 怎么做 |
|------|--------|
| **看到好文章没时间读** | 一键保存，关掉标签页，以后再看 |
| **收集参考资料** | 保存的页面自动带上标题和链接，按时间排序 |
| **整理阅读清单** | 双击重命名、编辑备注、标记重点 |
| **误删不用慌** | 回收站自动保留 30 天，随时恢复 |
| **边浏览边管理** | 打开侧面板，不离开当前页面就能查看和整理 |

## ✨ 为什么选择它？

- **⚡ 零操作成本** — 点击一次工具栏图标，再点一下"添加"，2 秒完成保存
- **📂 双模式操作** — Popup 弹窗快速添加，侧面板沉浸管理
- **🔄 误删可撤销** — 删除后 10 秒内一键恢复，手滑也不怕
- **🗑️ 回收站自动清理** — 30 天后自动清除，无需手动维护
- **🔒 完全离线，隐私安全** — 所有数据存在浏览器本地，不上传任何服务器，不需要账号，不注入网页脚本
- **📦 轻量无依赖** — 纯原生 JavaScript，不依赖任何框架，几乎不占用内存

## 📦 安装方式

### Chrome 网上应用店（推荐）

搜索 **Octoman Read Later** 或访问 Chrome 网上应用店安装。

### 开发者模式加载

1. 下载代码，打开 `chrome://extensions/`
2. 开启「开发者模式」→「加载已解压的扩展程序」
3. 选择 `read-later-extension` 文件夹

## 🚀 快速上手

### 保存当前页面

```
点击工具栏图标 → 点击「➕ 添加当前页面」→ 完成 ✅
```

### 查看和管理

- **Popup 弹窗**：点击工具栏图标，快速查看最近保存的页面
- **侧面板**：点击 📂 或右键扩展图标选择「在侧面板中打开」，边浏览边管理

### 编辑与整理

- **快速重命名**：双击标题直接编辑
- **完整编辑**：点击 ✏️ 修改标题、URL 和备注
- **打开链接**：点击 URL 在新标签页打开
- **删除**：点击 🗑️ 移入回收站（10 秒内可撤销）

### 回收站

- 点击 🗑️ 切换到回收站视图
- **恢复**：点击 ↩️ 恢复条目
- **永久删除**：点击 🗑️ 彻底删除
- **自动清理**：30 天后自动清除过期条目

## 🗂️ 项目结构

```
read-later-extension/
├── manifest.json          # 扩展清单
├── background.js          # Service Worker（数据管理、消息处理）
├── shared.js              # 共享工具函数
├── popup.html / popup.js  # Popup 弹窗
├── sidepanel.html / .js   # 侧面板
├── styles.css             # 全局样式
└── icons/                 # 扩展图标
```

## 🛠️ 技术栈

- **Manifest V3** — 最新 Chrome 扩展规范
- **Chrome Storage API** — 本地数据持久化
- **Chrome Side Panel API** — 侧面板支持
- **原生 JavaScript** — 无框架，轻量快速

## 🔒 隐私

- 所有数据**仅存储在本地浏览器**，不上传任何服务器
- 仅在用户主动操作时读取当前标签页的标题和 URL
- **无需注册、无需登录、不收集任何个人信息**
- **无需 `host_permissions`**，不会注入脚本到任何网页

## 📦 打包发布

```bash
cd read-later-extension
zip -r ../octoman-read-later-v1.0.zip . -x "*.DS_Store" ".DS_Store"
```

上传至 [Chrome 开发者控制台](https://chrome.google.com/webstore/devconsole) 即可发布。
